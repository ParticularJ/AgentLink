# 波段交易系统 - 第一阶段：赛道与个股初筛

基于政策+资金+产业三层漏斗的波段交易初筛系统。

## 核心特性

- **大盘环境过滤**：绿灯/黄灯/红灯三级评估体系
- **三层漏斗赛道发现**：政策驱动 + 资金驱动 + 产业驱动
- **产业链四层拆解**：上游/中游/下游/支撑层 + 景气传导时序
- **支撑层量化识别**：客户覆盖、耗材属性、设备属性、服务属性、国产替代空间
- **三因子评分模型**：产业动量(35%) + 个股弹性(35%) + 安全边际(30%)
- **板块轮动节奏匹配**：启动期/加速期/高潮期/衰退期
- **差异化档位判定**：第一档(核心) + 第二档(重点) + 第三档(观察)

## 六步流程

1. **大盘环境过滤** - 判断当前是否适合进行波段交易
2. **赛道发现** - 政策+资金+产业三层漏斗
3. **产业链四层拆解** - 上游/中游/下游/支撑层 + 景气传导时序
4. **龙头识别** - 各环节2-3家公司
5. **强制过滤** - 财务排雷 + 涨幅/流动性过滤
6. **三因子评分 + 档位判定** - 输出初筛股票池

## 输出文件

- `波段初筛池_YYYYQN.csv` - 核心输出（第一档+第二档）
- `产业链图谱_YYYYQN.md` - 辅助输出
- `催化日历_YYYYQN.md` - 事件驱动参考
- `大盘环境评估_YYYYQN.md` - 环境评估报告

## 使用方式

```bash
# 运行初筛
python scripts/main.py --quarter 2026Q2

# 测试
python tests/test_phase1.py
```

## 与第二阶段衔接

输出文件 `波段初筛池_YYYYQN.csv` 作为第二阶段（基本面二次筛选）的输入。
