# 波段交易系统

基于两份操作指导文档实现的波段交易筛选系统。

## 文档来源

- **第一阶段**: 波段交易系统 · 第一阶段：赛道与个股初筛操作指导 V5.0
- **第二阶段**: 波段交易系统 · 第二阶段：基本面二次筛选操作指导 V4.0

## 系统架构

```
band_trading_strategy/
├── phase1_initial/          # 第一阶段：赛道与个股初筛
│   ├── README.md
│   ├── config/
│   │   ├── market_config.py      # 大盘环境配置
│   │   ├── sector_config.py      # 赛道配置
│   │   └── scoring_config.py     # 评分配置
│   ├── scripts/
│   │   └── main.py               # 主入口
│   └── data/
│
└── phase2_secondary/        # 第二阶段：基本面二次筛选
    ├── README.md
    ├── config/
    │   ├── scoring_config.py     # 财报评分配置
    │   └── valuation_config.py   # 估值配置
    ├── scripts/
    │   └── main.py               # 主入口
    └── data/
```

## 第一阶段核心特性

- **大盘环境过滤**: 绿灯/黄灯/红灯三级评估
- **三层漏斗赛道发现**: 政策 + 资金 + 产业
- **产业链四层拆解**: 上游/中游/下游/支撑层 + 景气传导时序
- **三因子评分**: 产业动量(35%) + 个股弹性(35%) + 安全边际(30%)
- **差异化档位**: 第一档(核心) + 第二档(重点) + 第三档(观察)

## 第二阶段核心特性

- **差异化及格线**: 第一档>=35分，第二档>=40分
- **护城河定量评估**: 满分20分评分卡
- **一票否决**: 6项动态否决条件
- **财报评分**: 满分60分（含超预期和机构态度）
- **三轨估值**: PEG + PB-ROE + PS
- **历史估值分位点**: 相对位置判断
- **仓位计算**: 两系数模型
- **止损设定**: 技术止损 + 时间止损 + Beta调整

## 使用方式

### 第一阶段

```bash
cd phase1_initial
python scripts/main.py --quarter 2026Q2
```

输出:
- `波段初筛池_2026Q2.csv` - 核心输出
- `产业链图谱_2026Q2.md` - 辅助输出
- `催化日历_2026Q2.md` - 事件驱动参考
- `大盘环境评估_2026Q2.md` - 环境评估报告

### 第二阶段

```bash
cd phase2_secondary
python scripts/main.py --quarter 2026Q2 --input ../phase1_initial/波段初筛池_2026Q2.csv
```

输出:
- `待购候选池_2026Q2.csv` - 核心输出（约30-50只）
- `观察清单_2026Q2.md` - 待观察股票
- `财报评分明细_2026Q2.xlsx` - 评分底稿
- `估值分析底稿_2026Q2.xlsx` - 估值底稿

## 数据流

```
第一阶段初筛池 (60-80只)
    ↓ CSV传递
第二阶段二次筛选 (30-50只)
    ↓ CSV传递
第三阶段买入时机监控
```

## 否决条件分工

| 否决项 | 第一阶段 | 第二阶段 |
|--------|----------|----------|
| 经营现金流连续两季为负 | ✅ | — |
| 应收账款/营收 > 50% | ✅ | — |
| 商誉/总资产 > 30% | ✅ | — |
| 管理层违规记录 | — | ✅ |
| 审计意见非标 | — | ✅ |
| 北向资金持续流出 | — | ✅ |
| 大股东高比例质押 | — | ✅ |
| 近期大额解禁 | — | ✅ |
| 财报前涨幅过大 | — | ✅ |
