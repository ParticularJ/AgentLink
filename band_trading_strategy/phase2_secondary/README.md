# 波段交易系统 - 第二阶段：基本面二次筛选

对第一阶段初筛池进行深度基本面验证，输出可交易的候选池。

## 核心特性

- **差异化及格线**：第一档>=35分（放松），第二档>=40分（严格）
- **护城河定量评估**：定价权/差异化壁垒 + 客户粘性 + 利润含金量
- **一票否决验证**：管理层违规、审计意见、北向资金、质押、解禁、财报前涨幅
- **财报核心指标评分**：营收增速、利润增速、业绩趋势、现金含量、ROE
- **季报超预期评分**：实际业绩 vs 一致预期（波段核心催化剂）
- **机构态度验证**：评级上调/维持/下调
- **双轨估值体系**：PEG法 + PB-ROE法 + PS法（新增）
- **历史估值分位点**：当前估值在历史中的位置
- **仓位计算**：两系数模型（估值系数 × 护城河系数）
- **止损位设定**：技术止损 + 时间止损 + Beta调整

## 六步流程

1. **读取与排序** - 读取第一阶段CSV，按档位排序
2. **护城河定量评估** - 护城河评分卡（满分20分）
3. **一票否决验证** - 6项动态否决条件
4. **财报核心指标评分** - 满分60分（含超预期和机构态度）
5. **估值与波段空间判断** - PEG/PB-ROE/PS + 历史分位点
6. **综合决策矩阵 + 仓位计算** - 输出待购候选池

## 输出文件

- `待购候选池_YYYYQN.csv` - 核心输出（约30-50只）
- `观察清单_YYYYQN.md` - 待观察股票
- `财报评分明细.xlsx` - 评分底稿
- `估值分析底稿.xlsx` - 估值底稿

## 使用方式

```bash
# 运行二次筛选
python scripts/main.py --quarter 2026Q2 --input ../phase1_initial/波段初筛池_2026Q2.csv

# 测试
python tests/test_phase2.py
```

## 与第一阶段衔接

输入文件：`../phase1_initial/波段初筛池_YYYYQN.csv`

必需字段：代码、名称、第一阶段档位、第一阶段总分、赛道、产业链层级、卡位评级、Beta、板块轮动阶段、催化日历

## 与第三阶段衔接

输出文件 `待购候选池_YYYYQN.csv` 作为第三阶段（买入时机监控）的输入。
