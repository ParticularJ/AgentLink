# 触发规则

**收到以下任务时调用：**
- 写讲话、写发言稿、写汇报材料、写致辞、生成讲话稿

---

# 执行流程

## 1. 选择风格界面

```
请选择风格：

A. 已有领导风格
   - 张局

B. 冷启动模板
   - 务实型 / 激情型 / 儒雅型 / 严谨型

C. 新增领导风格
   - 上传3-5篇讲话稿，自动提取风格

请选择：A / B / C
```

## 2. 确认素材
- 有素材 → 提取关键信息
- 无素材 → 查素材库
- 无论有无素材 → 查金句库 → 网络搜索补充最新信息

## 3. 搜索补充
- 使用tavily-search搜索最新政策/数据/金句

## 4. 生成初稿
- 标注数据来源

## 5. 输出
- docx文件
- md文件

# 金句库

**位置：** `speech-writer/gold-sentences/`

**功能：**
- 查询金句：搜索相关主题金句
- 添加金句：搜索后确认存入
- 自动提取：新增领导风格时自动提取

**金句识别规则：**
- 有引号："xxx"
- 诗词典故："周公吐哺，天下归心"
- 四字格短句
- 引用词：指出、强调、说过

**加入流程：**
初稿完成后，发现好金句时提示：
```
发现以下金句，要加入金句库吗？
1. "xxx" - 来源

A. 确认加入
B. 不加入
```

# 素材库

**位置：** `speech-writer/materials/`

**存储内容：**

- 网络搜索补充的内容
