# Research Report Creator - 行业产业链深度调研报告生成工具

**版本**: v2.0  
**适用行业**: 科技制造、医药生物、新能源、周期资源、消费服务、高端装备  
**报告格式**: 光通信/PCB/电力电网产业链研报标准格式

---

## 📋 功能概述

本 Skill 用于生成专业的行业产业链深度调研报告，包含：

- ✅ 产业链全景结构分析（上游/中游/下游/支撑层）
- ✅ 各环节详细分析（趋势+数据+竞争格局）
- ✅ 投资建议汇总（按方向精选龙头+潜力标的）
- ✅ 投资标的深度分析（核心竞争力/成长性/估值/风险）
- ✅ 全产业链总结与投资策略
- ✅ 投资逻辑总结与风险提示
- ✅ 可视化图表数据（JSON格式）
- ✅ 财务深度分析（可选）

## 📊 报告结构

### 一、产业链全景结构
- 1.1 产业链层级图（文字层级展示）
- 1.2 上游分析（材料/设备）
- 1.3 中游分析（制造/设计）
- 1.4 下游分析（模组/应用）
- 1.5 支撑层分析（设备/EDA）
- 1.6 产业链核心龙头汇总表

### 二、投资建议汇总（6大方向）
- 方向1：核心主线（如AI服务器存储）
- 方向2：国产替代（如HBM）
- 方向3：卖铲人-材料
- 方向4：卖铲人-设备
- 方向5：稳健增长
- 方向6：封装测试

### 三、投资标的深度分析（8-12只）
每只标的包含：
- 【核心竞争力】产业链卡位+技术壁垒+客户资源
- 【成长性】短期（1-2年）+长期（3-5年）
- 【估值分析】PE/PB/PS+机构目标价
- 【风险提示】行业共性风险+标的个性风险

### 四、全产业链总结与投资策略
- 4.1 核心标的排序（第一/二/三梯队）
- 4.2 产业链投资地图（按环节配置比例）
- 4.3 一句话策略（比喻式总结）

### 五、投资逻辑总结与风险提示
- 5大核心投资逻辑
- 6大核心风险提示

### 六、数据来源
- 一级信源（公司财报）
- 二级信源（行业咨询/券商研报）
- 三级信源（财经媒体）

## 🎯 报告特色

### 1. 标准化格式
- 统一采用光通信/PCB/电力电网产业链研报格式
- 各环节必须包含：关键趋势+细分表格+关键数据

### 2. 量化分析
- 所有投资逻辑必须含量化数据
- 所有风险提示尽量量化影响（如"毛利率将下降约20-30%"）

### 3. 产业链思维
- 从上游到下游完整梳理
- 标注各环节成本占比和利润池分布
- 明确技术壁垒等级（高/中/低）

### 4. 投资导向
- 按方向精选投资标的
- 区分龙头（确定性）和潜力（高弹性）
- 提供配置比例建议

## 🔧 数据源等级体系

| 等级 | 定义 | 通用特征 |
|------|------|---------|
| **一级** | 官方/公司/核心客户 | 不可篡改、一手、可审计 |
| **二级** | 顶级投行/行业咨询/头部券商 | 方法论成熟、数据可追溯 |
| **三级** | 商业咨询/财经媒体/数据平台 | 仅用于事件验证或交叉核对 |
| **禁止** | 自媒体/论坛/传言 | 任何未经溯源的信息 |

### 行业数据源映射

| 行业类型 | 一级信源核心 | 二级信源核心 | 三级信源参考 |
|---------|------------|------------|------------|
| **科技制造** | 公司财报、云厂商capex、海关数据 | LightCounting/Yole/Prismark、头部券商 | 中商产业研究院、C114通信网 |
| **医药生物** | NMPA审评数据、临床公告、医保谈判 | Frost & Sullivan/Evaluate Pharma | 医药魔方、Insight数据库 |
| **新能源** | 能源局装机数据、电网招投标 | BloombergNEF/IEA | 智研咨询、北极星电力网 |
| **周期资源** | 国家统计局产量数据、期货库存 | CRU/Wood Mackenzie | 百川盈孚、生意社 |
| **消费服务** | 公司门店数据、社零统计 | 欧睿/Nielsen | 艾瑞咨询、红餐网 |
| **高端装备** | 国防预算/航天计划、军品订单 | Teal Group/SIPRI | 航宇微、集微网 |

## 📝 报告格式规范

### 标题格式
```
[产业链名称]产业链全景图与投资标的深度分析
[年份]年版 · 含产业链全景、投资建议及标的深度分析
核心逻辑：[驱动因素A] × [驱动因素B] × [行业特征]
```

### 标的编号格式
```
5.X 公司名称（股票代码）—— 定位标签

定位标签示例：
- 全球[产品]绝对龙头 · 核心龙头
- [差异化特征] · 潜力龙头
- [技术突破点] · 高弹性卫星
- [卖铲人特征] · 防守配置
- [反转逻辑] · 困境反转
```

### 数据引用格式
```
据[机构]预测，2026年[指标]达[数值][citation:X]
```
