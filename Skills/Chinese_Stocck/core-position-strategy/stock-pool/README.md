# 核心仓股票池

核心仓策略的独立股票池，包含初筛池和候选池。

## 文件说明

| 文件 | 说明 |
|:---|:---|
| `watchlist.yaml` | 核心仓观察列表 |
| `watchlist_copy.yaml` | 观察列表备份 |
| `initial_pool_YYYYQ#.csv` | 季度初筛池（50-60只） |
| `candidate_pool_YYYYQ#.csv` | 季度候选池（5-8只） |

## 数据流

```
季度选股流程
    ↓
初筛池（50-60只）
    ↓ 财报验证
候选池（5-8只）
    ↓ 每日买点判断
买入持仓
```

## 更新频率

- **初筛池**：每季度首月更新
- **候选池**：财报季更新
- **观察列表**：持续维护
