# 热点仓股票池

热点仓策略的独立股票池，包含涨停板股票和热点板块标的。

## 文件说明

| 文件 | 说明 |
|:---|:---|
| `limit_up_pool.json` | 涨停股票池 |
| `sector_leaders.json` | 板块龙头池 |
| `new_stocks.json` | 次新股池 |
| `retrace_candidates.json` | 回调候选池 |

## 数据流

```
每日市场扫描
    ↓
涨停股票池
板块龙头池
次新股池
    ↓
四大策略评分
    ↓
Top 3 推荐
```

## 更新频率

- **实时更新**：交易时段每15分钟
- **收盘更新**：每日收盘后完整更新
