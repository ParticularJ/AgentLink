# 策略工具

跨策略通用的分析工具、优化器和融合顾问，为三大策略提供数据支持和决策辅助。

## 包含工具

| 工具 | 说明 | 功能 |
|:---|:---|:---|
| **strategy-fusion-advisor** | 策略融合顾问 | 多策略信号融合、冲突解决、综合评分 |
| **strategy-optimizer** | 策略优化器 | 策略参数优化、回测、绩效分析 |
| **stock_earnings_analysis** | 财报分析工具 | 财报数据抓取、超预期检测、业绩评分 |
| **stock_event** | 股票事件监控 | 公告、新闻、政策事件实时监控 |

## 功能说明

### 策略融合顾问
- 接收多个策略的交易信号
- 分析信号一致性/冲突
- 综合评分输出最终建议
- 仓位分配优化

### 策略优化器
- 历史数据回测
- 参数敏感性分析
- 最优参数搜索
- 策略绩效报告

### 财报分析工具
- 自动抓取财报数据
- 计算关键财务指标
- 检测业绩超预期
- 生成财报评分

### 股票事件监控
- 实时监控公司公告
- 跟踪新闻舆情
- 政策影响分析
- 事件驱动信号生成

## 使用方式

各工具独立运行，详见各目录下的文档。
