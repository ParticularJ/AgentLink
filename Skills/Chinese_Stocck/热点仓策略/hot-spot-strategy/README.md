# 热点仓完整交易策略 (Hot Spot Strategy) V1.1

## 📊 策略概述

热点仓是整个交易体系中专门用于超短线交易的独立资金模块，与波段仓、底仓并行运作，互不干扰。

**目标**：捕捉A股短期情绪溢价和资金惯性
**持股周期**：3-5个交易日（龙头股可延长至8天）
**资金占比**：总资金的20%-30%

## 📖 策略原理

### 1. 首次涨停板策略（策略一）

**核心逻辑**：捕捉个股从量变到质变的突破节点，依赖涨停板制度下的流动性突变和价格发现机制。

**评分模型（表3）**：
| 维度 | 权重 | 标准 |
|------|------|------|
| 涨停时间 | 30分 | 9:35前30分；10:00前20分；10:30前10分 |
| 封单质量 | 25分 | 封单/日均成交额>80%得25分；>50%得20分；>30%得15分 |
| 板块效应 | 25分 | 板块内≥5只涨停25分；≥3只15分；≥1只5分 |
| 换手率 | 20分 | 5%-15%得20分；15%-25%得15分；3%-5%得10分 |

**一票否决**：一字板、大盘破20日线、个股有利空

**买入时机**：当日打板（排板或扫板）或次日早盘集合竞价

**仓位建议（表4）**：
| 分数区间 | 评级 | 仓位 |
|----------|------|------|
| 90-100分 | 极强势 | 25%-30% |
| 80-89分 | 强势 | 20%-25% |
| 75-79分 | 中等 | 15%-20% |

### 2. 涨停回调策略（策略二）

**核心逻辑**：基于行为金融学中的"过度反应修正"原理，利用获利回吐后的技术性反弹获取收益。是整个体系的"压舱石"。

**评分模型（表5）**：
| 维度 | 权重 | 标准 |
|------|------|------|
| 回调幅度 | 25分 | -8%~-12%得25分；-5%~-8%得20分；-12%~-15%得15分 |
| 回调天数 | 20分 | 3-4天得20分；2天或5天得15分 |
| 缩量程度 | 25分 | 量能<涨停日40%得25分；<60%得15分 |
| 企稳信号 | 30分 | 收阳+站5日线+跌幅≤-3%后拉起得30分 |

**一票否决**：大盘破20日线、板块退潮、个股利空

**最佳买入时机**：出现企稳信号的当日尾盘（14:50后）或次日早盘

**加仓规则**：浮盈≥5% + 持有≤3天 + 分时弱转强 + 板块回流，加仓20%-30%

**仓位建议（表6）**：
| 分数区间 | 评级 | 仓位 |
|----------|------|------|
| 90-100分 | 极强势 | 25% |
| 80-89分 | 强势 | 25% |
| 70-79分 | 中等 | 20% |
| 65-69分 | 及格 | 15% |

### 3. 热门板块龙头策略（策略三）

**核心定义**：龙头股 = 板块内阶段涨幅领先 + 带动板块 + 资金认可

**板块热度评分（表8）**：
| 指标 | 权重 | 标准 |
|------|------|------|
| 板块涨幅 | 30% | 涨幅排名前10%得100分 |
| 涨停家数 | 25% | ≥5只得100分 |
| 资金净流入 | 20% | 占成交额比>5%得100分 |
| 趋势强度 | 15% | 站20日线且向上得100分 |
| 连板高度 | 10% | ≥3连板得100分 |

**龙头股评分（表9）**：
| 指标 | 权重 | 标准 |
|------|------|------|
| 涨幅领先度 | 25分 | 近5日涨幅板块第1得25分 |
| 涨停强度 | 15分 | 连续涨停得15分 |
| 板块热度 | 20分 | 板块总分≥85分得20分 |
| 带动性 | 15分 | 该股上涨时板块明显跟涨得15分 |
| 换手率 | 15分 | 10%-25%得15分 |
| 流通市值 | 10分 | 50亿-150亿得10分 |

**龙头买点类型（表10）**：
| 龙头类型 | 买点条件 | 买入动作 |
|----------|----------|----------|
| 连板龙头首次分歧 | 连板后首次开板，盘中跌幅≤-3%后回升 | 尾盘买入25% |
| 连板龙头弱转强 | 次日低开后30分钟内翻红站均线 | 盘中加仓或新开 |
| 趋势龙头回调均线 | 沿5日线上行，首次回踩5日线不破 | 尾盘买入30% |
| 趋势龙头平台突破 | 横盘2-5天后，放量上涨≥5% | 盘中买入30% |
| 趋势龙头板块分歧日抗跌 | 板块调整时该股跌幅<板块1/2 | 尾盘买入25% |

### 4. 次新股策略（策略四）

**核心逻辑**：适用标的为上市当日的新股（前5日不设涨跌幅），风险最高、仓位最严格。

**一票否决（表12）**：
- 首日换手率 > 80%（标准盘）/ > 90%（小盘）→ 抛压过重
- 首日换手率 < 50%（标准盘）/ < 70%（小盘）→ 惜售
- 尾盘仍涨幅 > 100% → 透支空间
- 尾盘仍在创新低 → 无企稳
- 龙虎榜卖方全是机构 → 机构在卖
- 大盘破20日线 → 系统性风险

**评分模型（表13）**：
| 维度 | 权重 | 标准 |
|------|------|------|
| 行业景气度 | 30分 | 当期主线核心赛道得30分 |
| 首日换手率 | 25分 | 标准盘60%-80%得25分；小盘70%-90%得25分 |
| 尾盘企稳 | 25分 | 尾盘30分钟站稳分时均线且不创新低得25分 |
| 发行估值 | 20分 | 发行PE≤行业PE×1.2得20分 |

**唯一买点**：上市当日尾盘（14:50-15:00），禁止盘中任何时点买入

## 💰 资金管理

### 核心规则

| 规则 | 数值 | 说明 |
|------|------|------|
| 热点仓占总资金 | 20%-30% | 根据市场环境动态调整 |
| 单只股票最大仓位 | ≤热点仓资金的30% | 避免单票风险过度集中 |
| 同时持有股票数量 | ≤3只 | 确保每只标的都有足够的盯盘精力 |
| 单日总亏损上限 | 热点仓资金的2% | 触发二级熔断强制清仓 |

| 90-100分 | 极强势 | 25%-30% |
| 80-89分 | 强势 | 20%-25% |
| 75-79分 | 中等 | 15%-20% |
| 65-69分 | 及格 | 10%-15% |

## 🛡️ 风险控制

### 止损体系

| 止损类型 | 规则 | 适用 |
|----------|------|------|
| 固定止损 | 成本价-7% | 所有策略 |
| ATR动态止损 | 买入价-2×ATR(14) | 所有策略 |
| 移动止损 | 从最高点回撤-5% | 浮盈>5%后启用 |
| Chandelier Exit | N日最高价-3×ATR(14) | 趋势股持仓跟踪 |
| 时间止损 | 持有5日未盈利 | 所有策略（次新股3天） |
| 均线止损 | 收盘破5日线 | 所有策略 |

### 止盈体系

**普通热点股**：
| 浮盈区间 | 操作 | 剩余仓位处理 |
|----------|------|--------------|
| 5%-10% | 卖出1/3 | 移动止损-5% |
| 10%-20% | 再卖1/3 | 移动止损-3% |
| >20% | 剩余保本单 | 博取超额 |

**龙头股**：
| 浮盈区间 | 操作 | 剩余仓位处理 |
|----------|------|--------------|
| 5%-10% | 卖出1/4 | 移动止损-6% |
| 10%-20% | 再卖1/4 | 移动止损-4% |
| 20%-30% | 再卖1/4 | 移动止损-3% |
| >30% | 剩余1/4 | 保本单博取超额 |

### 熔断机制

```
日亏损达1.5% → 一级熔断：禁止开新仓
日亏损达2.0% → 二级熔断：强制清仓，停止交易
连续2日二级熔断 → 暂停交易3天，深度复盘
```

## 📈 监控体系

### 盘中实时监控（每5-15分钟）

| 监控项 | 正常状态 | 警报状态 | 行动 |
|--------|----------|----------|------|
| 分时走势 | 沿分时均线上行 | 跌破分时均线>15分钟 | 卖出1/3 |
| 量能 | 温和放量 | 爆量(+100%以上且不涨) | 卖出1/2 |
| 板块情绪 | 板块红盘 | 板块内出现≥3只跌停 | 清仓该板块 |
| 封板状态 | 正常封板 | 炸板>10分钟未回封 | 清仓该股 |
| 龙头地位 | 涨幅板块前5 | 被其他股超越 | 减仓50%或切换 |

### 收盘后每日监控

| 监控项 | 健康标准 | 危险信号 | 次日行动 |
|--------|----------|----------|----------|
| 均线 | 收盘>5日线 | 跌破5日线 | 开盘卖出 |
| 量价 | 上涨放量、下跌缩量 | 高位放量滞涨 | 开盘卖1/2 |
| K线 | 实体阳线 | 长上影(＞实体2倍) | 反弹无力即卖 |
| 换手率 | 10%-25% | >35% | 冲高卖出 |
| 龙头评分 | ≥70分 | 连续2日下降 | 减仓50% |
| 板块热度 | ≥70分 | 退出热度前5 | 清仓该板块 |

## ⏰ 每日操作流程

| 时间 | 工作内容 | 重要程度 |
|------|----------|----------|
| 盘前 8:30-9:00 | 检查隔夜消息、调整计划 | 高 |
| 早盘 9:30-10:30 | 执行买入、实时监控、止损/止盈 | 极高 |
| 盘中 10:30-14:30 | 持仓监控、策略扫描 | 中 |
| 尾盘 14:30-15:00 | 执行尾盘买入、确认持仓状态 | 高 |
| 盘后 15:30-17:00 | 每日复盘、计算评分、确定次日候选 | 高 |
